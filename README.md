**ReflexLab** is a premium, high-performance reaction trainer designed for competitive gamers and aiming enthusiasts. Built with a sleek Cyberpunk aesthetic, it provides an optimized training environment to sharpen your motor skills and track your progress over time.

---

## 🚀 Features

### 🕹️ Training Modes
*   **Static Click**: The classic reaction test. Click targets as they appear.
*   **Moving Target**: Track and hit moving objects to master dynamic aiming.
*   **Precision Mode**: Targets are smaller and require higher accuracy.
*   **Time Attack**: How many hits can you land in 30 seconds?

### 📊 Real-time Analytics
*   **Performance Dashboard**: Live stats for Hits, Misses, and Accuracy.
*   **Reaction Tracking**: Measures average and best reaction times in milliseconds.
*   **Progress Charts**: Integrated visualizer to see your improvement across sessions.

### 🎨 Visuals & Sound
*   **Neon Aesthetic**: Deep space background with cyan/red neon glow effects.
*   **Adaptive Audio**: Procedural sound effects synced to hits and misses.
*   **Glassmorphism UI**: Modern, translucent UI components with blur effects.

---

## 🛠️ Tech Stack

*   **Frontend**: [React 19](https://react.dev/)
*   **Build Tool**: [Vite](https://vitejs.dev/)
*   **Graphics**: [HTML5 Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) for high-performance rendering.
*   **Charts**: [Chart.js](https://www.chartjs.org/) & [react-chartjs-2](https://react-chartjs-2.js.org/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **Styling**: Modern CSS with Variables and Glassmorphism.

## ⚡ Getting Started

### Prerequisites
*   [Node.js](https://nodejs.org/) (v18+)
*   npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/remilsalim/ReflexLab.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
